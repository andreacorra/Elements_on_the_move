# Elements on the move
This repository contains the code needed to reproduce the analysis of the manuscript 'Elements on the Move: How Ungulate Migration and Movement Expands Alpine Biogeochemical Footprints' (202X)

Authors: Kristy M. Ferraro, Andrea Corradini, Olivier Chanel, Charles Figuières, Yann Bramoullé, Vivian Hawkinson, Charlotte Vanderlocht, Federico Ossi, Davide Righetti, Oswald J. Schmitz, Francesca Cagnacci
